About

What is this?
This is a Mac OS X Leopard portal (which means it's a set of offline-webpages that looks like the real Mac OS X Leopard). It has some useful features, such as games and applications. You do not need an network connection to use this portal, if you're not going to use Safari, iChat, and Update. The most of its features only need to use 1 single tab in the PSP's internet browser, which means you can handle more portals at the same time (if you don't get memory errors).
This Mac OS X Leopard portal is based off of the real Mac OS X Leopard made by Apple Corporation.
