What is pspWvista?

pspWvista allows you to use Windows Vista on your Playstation Portable. Well, really it is just a set of webpages that look similar to Windows Vista, but they are offline. It has some useful features: lots of games and applications. You do not need to access the internet for most of pspWvista, only to access the links and use the web browser. This way it's much faster.

pspwvista is based off of pspWxp 3.7 and is actually pspWxp 3.9.

LEGAL Information:

This is NOT homebrew - it is a set of webpages. It will not hurt you PSP in any way. It's just like a website but it is offline, meaning you do not need to be connected to the internet to use it. pspWxp just opens in the internet browser. It is free and totally LEGAL.
